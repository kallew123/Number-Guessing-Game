# 🎯 Number Guessing Game (Python)

A command-line number guessing game developed in Python.

The player must guess a randomly generated number within a limited number of attempts. The game provides feedback after each guess, indicating whether the correct number is higher or lower.

## 🚀 Features
- Multiple difficulty levels:
  - Easy (1–50)
  - Medium (1–100)
  - Hard (1–200)
- Limited attempts (5 tries)
- Hints after each guess (higher or lower)
- Input validation (handles non-numeric input)

## 🧠 Concepts Used
- Random number generation (`random.randint`)
- Loops (`for`, `while`)
- Conditionals (`if/elif/else`)
- Functions
- Exception handling (`try/except`)
- User input handling
